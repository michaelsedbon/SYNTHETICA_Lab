# EXP_004: Automated Connector Mapping

**Start Date:** 2026-03-03
**Status:** In progress
**Airtable Links:** None

---

## Overview

Automated connector mapping tool for the Cryptographic Beings machine. Uses an Arduino Mega with a sensor shield to scan continuity between 16 connectors on Panel A (front 4×4 grid) and 16 connectors on Panel B (top plate), replacing manual multimeter probing.

## Goal

Determine which connector on Panel A maps to which connector on Panel B by automated continuity scanning, with results displayed in a real-time web UI.
