Quick Start

  1. Click Connect and select the Arduino Mega serial port
  2. You should see a pong response confirming the connection
  3. Wire Panel A connectors to pins D22–D37 and Panel B connectors to pins D38–D53
  4. Click Scan All to map all connectors automatically
  5. Click any Panel A connector circle to test a single connector
  6. Click Export JSON to download the mapping
