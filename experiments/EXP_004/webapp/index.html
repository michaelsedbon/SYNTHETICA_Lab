📖 Quick Start
Wiring (8 wires to Arduino Mega):
Aviator cable → D22(Red) D23(Black) D24(White) D25(Blue)
M12 cable → D38(White) D39(Blue) D40(Black) D41(Red)
M8 cable → D38(Blue) D39(Black) D40(White) D41(Red)
Workflow:
1. Click Connect → select Arduino serial port
2. Plug M12/M8 into a p position → select it in the dropdown
3. Click ▶ Start Scanning
4. Try plugging the aviator cable into L1, L2, L3… until you hear a 🔊 beep
5. Select which L and click ✓ Confirm
6. Move M12/M8 to next p → repeat
7. Export when done — or Import a previous file to review/correct
