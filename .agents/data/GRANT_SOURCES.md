# Grant Funding Sources Registry

> Last updated: 2026-03-17
> Total sources: 30

This file is the source of truth for known funding databases and listing sites.
The `/search-grants` workflow reads this file each run and searches sources that are due.

## How to read this file

Each source entry has:
- **Status**: `active` | `inactive` | `broken` | `new`
- **Frequency**: How often to check — `weekly` | `biweekly` | `monthly`
- **Last checked**: Date of last search run (ISO format)
- **Region**: Geographic scope
- **Category**: `art` | `science` | `design` | `interdisciplinary` | `aggregator`
- **Requires account**: Whether login is needed (limits automated searching)
- **Collaboration-friendly**: Whether calls commonly welcome artist+scientist teams
